*****
SETUP
*****

These files enable you to import the TEI P5 sources of the OGR corpus into
TXM 0.8.1. 

1. Download the TEI P5 sources from http://www.ogr-corpus.org/docs/download
and unzip the archive.
2. Copy the contents of this .zip archive into the "tei" folder.
3. Launch TXM 0.8.1.
4. Launch the import "XML TEI Zero + CSV", select the "tei" folder, name
the corpus and *disable* tokenization.
5. Import the corpus.

These files are provided for advanced users who wish to modify the corpus 
and/or the edition and re-import it. Most users should just download the
pre-compiled TXM corpus from http://www.ogr-corpus.org/docs/download .
